Using Advanced Edit Mode
########################

Many edit tools have an advanced mode that provides access to additional edit features.

To try this for yourself:

1. Open up the application preferences using **Window > Preferences**
 2. Navigate to the `Edit Tool Preferences <Edit%20Tool%20Preferences.html>`_ page
 3. Toggle the **Advanced Tools** checkbox
 4. Press OK

Tip: You can also toggle this setting while editing by pressing CTRL+space.
