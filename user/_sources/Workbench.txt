Workbench
#########

The term Workbench refers to the desktop application. The Workbench aims to achieve seamless
integration and controlled openness by providing a common paradigm for the creation, management, and
navigation of spatial resources.

Each Workbench window contains views and editors and control what appears in certain menus and tool
bars. More than one Workbench window can exist on the desktop at any given time.
