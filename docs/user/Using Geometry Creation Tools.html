Using Geometry Creation ToolsÂ¶

Geometry Creation Tools are used to create or draw a new Geometry. This geometry is used as the starting point when creating a new feature (you will sill need to fill in the attributes).

Polygon Creation Tool

Line Creation Tool

Point Creation Tool

Draw Geometry Tool

Rectangle Tool

Ellipse Tool

Smart Buffer Tool

Tip: When Advanced Edit Mode is active many of these geometry creation tools can also be used for editing existing features as well. This is convenience to allow you to quickly correct your data entry work while reducing the amount of times you need to switch tools.
