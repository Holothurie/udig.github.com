View Error Log and Configuration DetailsÂ¶

To review the uDig application log file in notepad (or your favourite text editor).

  1. Choose Help > About uDig from the menu bar
  2. Press the Configuration Details button, this screen shows all kinds of platform specific information
  3. Press the View Error Log button
  4. Choose the program you want to use to open the file from the list, I usually use Notepad when on windows

You will want to scroll the the bottom of the file to have a look at recent activity.
