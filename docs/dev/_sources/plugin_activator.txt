Plugin Activator
================

The **Activator** class is generated by the :doc`creating_a_plugin` and is responsible for managing 
the life cycle of your plugin.

The focus of the Activator class is Resource Management:

*  Internationalization support via :doc:`plugin_internationalization_with_resourcebundles`
*  Management of system resources using utility classes like ImageCache, FontCache and so on
